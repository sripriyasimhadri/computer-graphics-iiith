The Home page has description of the experiment and buttons which redirect to display and world pages respectively.

The world page has fields to enter X and Y coordinates and the width of diagonal of point displayed.

It also has a field to manipulate axis leghth max being 20 due to canvas size.

The drop down button transformtion enables user to transform point along X, Y and Z axes respectively.

The display page has the following options:

a.3D buttons shows three dimensional coordinates.

b.Show animation option tranforms point along Z axis.

c.Draw XY,YZ and ZX grids draws a grid on canvas.

d.Show units option shows the units of point.

e.draw option draws the coordinate axes.
